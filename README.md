# 🛒 Big Mart Sales Prediction Using Machine Learning

## 📌 Project Overview

This project predicts the sales of products sold in Big Mart stores using Machine Learning techniques. The model analyzes historical sales data and predicts the expected sales for different products based on their features.

This project was completed as part of the **Artificial Intelligence Internship** at **Naviotech Solution**.

---

## 🎯 Objectives

* Predict product sales using Machine Learning.
* Perform data preprocessing and cleaning.
* Handle missing values.
* Encode categorical features.
* Train and evaluate the XGBoost Regressor model.
* Improve business decision-making through predictive analytics.

## 📂 Dataset

The project uses two dataset files:

* Train.csv
* Test.csv

The target variable is:

**Item_Outlet_Sales**

---

## 🤖 Machine Learning Model

**XGBoost Regressor**

The model was trained using the training dataset and evaluated using the **R² Score**.

---

## 📊 Results

* Training R² Score: **0.8762**
* Testing R² Score: **0.5017**

The model successfully predicts product sales based on historical retail data.
